O Harmony OS desenvolvido pela Huawei tem grande semelhança com o android, aos usuários mais familiarizados seu uso sera mais simples

## Tela inicial

A tela inicial mostra os aplicativos abertos atualmente, os aplicativos favoritos, e alguns atalhos no canto inferior direito para [Configurações](/apps/config) e Para desligar, também um atalho para o Windows no canto inferior esquerdo, Passando para o lado direito temos a [Lista de aplicativos](#lista-de-aplicativos), e passando para o lado esquerdo temos a [Welcome Screen](#welcome-screen)

<img src="/images/start-screen.jpg" alt="Tela Inicial"/>

## Lista de aplicativos

A lista mostra todos os aplicativos instalados visíveis, é possível ocultar aplicativos nas configurações.

<img src="/images/app-grade.jpg" alt="Lista de aplicativos"/>

## Welcome Screen

A Welcome Screen é uma tela de apresentação, pode ter seu fundo e texto alterados, e funciona como um descanso de tela
