## Painel de controle

<img src="/images/shortcuts.png"/>

1. Configurações
2. Home (volta para tela inicial)
3. Microphone (controla se esta ligado ou desligado)
4. Camera (controla se esta ligado ou desligado)
5. Input Source (Controla entrada do conteúdo, IdeaShare ou HDMI)
6. Annotate (Da um print da tela atual e abre o Whiteboard)
7. Multi-window (Permite usar 2 aplicativos ao mesmo tempo)
8. Windows (Abre o sistema Windows do OPS)
9. Statusbar (Mostrar o código de conexão do IdeaShare)
10. Controle de brilho
11. Controle de Volume
15. Configurações
16. Desligar

## 13. Apps

Mostra os aplicativos frequentes (podem ser configurados) e clicando em *More* abre uma pasta com todos os aplicativos

<img src="/images/shortcut-apps.jpg"/>

### 14. Tasks

Mostra os aplicativos abertos recentemente, clicando em *Total* abre uma pasta com todos e a opção de fechar todas

<img src="/images/shortcut-tasks.jpg"/>
