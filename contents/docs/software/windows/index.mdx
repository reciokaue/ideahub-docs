O Ideahub possui um [OPS (Open Pluggable Specification)](/hardware/ops), uma plataforma x86 integrada que atualmente roda o Windows 11. Isso permite utilizar o equipamento como um PC convencional, com acesso completo ao sistema operacional da Microsoft, ideal para apresentações, reuniões, uso de softwares corporativos e mais.

### Como acessar o Windows no Ideahub

1. **Ligue o Ideahub** e aguarde o carregamento do sistema Harmony OS.
2. **Abra o [Painel de control](/software/harmonyos/control-painel)** clicando no botão na lateral do dispositivo.
3. **Clique no icone do Windows** e faça login no sistema

---

<Note title='Não desligue o Windows' type='danger'>
  Não desligue diretamente o Windows, caso feito, no proximo login será necessário liga-lo manualmente veja como em [Hardware OPS](/hardware/ops)
</Note>
